This component is still under development

List pickers are the new Select2. They are meant to be easier to use and to customise while having a similar UX.

Options:

  • placeholder (default: ""): A string message inviting the user to fill up the field
  • is_filterable (default: false): A boolean to add a search input in the list-picker dropdown
  • items_template_formatter (default: none): An asynchronous callback function returning a string to customise items templates
    It takes two parameters:
    • option_value: string The value of the current select box option
    • option_label: string The text content of the current select box option
