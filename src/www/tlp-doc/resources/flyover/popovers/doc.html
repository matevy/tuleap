tlp.createPopover(trigger, element, option = {});

Arguments:

  • { Element } trigger
  • { Element } popover content
  • (Optional) { Object } options. Contains:
    • { String } placement, defaults to "bottom". See examples below for available placements. Overrides corresponding data-placement attribute.
    • { String } trigger, defaults to "hover". If "click", popover is displayed on click instead of mouseover. Overrides corresponding data-trigger attribute.
    • { Element } anchor, defaults to trigger parameter. The arrow of the popover will be attached to the anchor.
